# Datasets

This directory stores benchmark datasets used for evaluating Large Language Models (LLMs).

## Planned Benchmarks

* GSM8K (Mathematical Reasoning)
* TruthfulQA (Hallucination Detection)
* HellaSwag (Commonsense Reasoning)

## Dataset Usage

Datasets are used to:

1. Evaluate model performance
2. Measure reasoning capabilities
3. Detect hallucinations
4. Compare benchmark results

## Notes

Benchmark datasets may be downloaded from their official sources and loaded through scripts in the `src/` directory.
