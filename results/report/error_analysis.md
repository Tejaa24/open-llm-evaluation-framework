# Error Analysis

## GSM8K Evaluation Observations

### Common Failure Modes

1. Multi-step arithmetic reasoning errors
2. Intermediate calculation mistakes
3. Numerical consistency issues
4. Incorrect final answer extraction

### Example Error

Question:
A train travels 60 km in 1 hour. How far does it travel in 3 hours?

Expected:
180

Predicted:
160

Reason:
Arithmetic reasoning failure.

### Research Insight

Even when language generation quality is high, numerical reasoning remains a challenging area for smaller language models.
